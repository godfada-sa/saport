I am a Full-Stack Engineer and Cybersecurity Specialist dedicated to building digital products that are secure, scalable, and exceptionally crafted. My work bridges the gap between innovative software development and advanced security engineering, ensuring every solution is designed for both performance and resilience.

From architecting modern web applications and cloud infrastructures to conducting penetration tests and vulnerability assessments, I approach every project with a dual perspective: creating systems that deliver outstanding user experiences while remaining resilient against real-world threats.
